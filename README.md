## USAGE


### Simple Usage
    $("#search").placeHolder();

### Specifying Placeholder Text
    $("#search").placeHolder({
        "text": "Your Placeholder Text"
    });

### All Options

    $("#search").placeHolder({
        "text"		 : "Your Placeholder Text",
        "placeholder": "#333333", /* Placeholder Text Color */
        "active"	 : "#000000" /* Normal Text Color */
    });
